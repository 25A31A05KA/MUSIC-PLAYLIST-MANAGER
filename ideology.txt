🎵 MUSIC PLAYLIST MANAGER – PROJECT IDEOLOGY

This project is designed to simulate a basic music playlist system using a Doubly Linked List in C++. The main idea behind this project is to demonstrate how fundamental data structures like linked lists can be applied to solve real-world problems such as managing and navigating a music playlist.

In modern music applications, users can easily move forward and backward between songs, add or remove tracks, and maintain a dynamic playlist. To replicate this behavior efficiently, a Doubly Linked List is used, where each node represents a song and contains links to both the next and previous songs. This allows smooth navigation in both directions.

The project focuses on dynamic memory management, efficient insertion and deletion of songs, and user-friendly interaction through a menu-driven interface. Unlike static data structures like arrays, the linked list provides flexibility in handling playlists of varying sizes without memory wastage.

Key functionalities include adding songs, deleting songs, searching for songs, displaying the playlist, and navigating between songs (next and previous). The concept of a "current playing song" is also maintained to mimic real-world playlist behavior.

The main objective of this project is to strengthen understanding of linked lists, pointers, and dynamic data handling in C++, while also building a practical and interactive application. It bridges the gap between theoretical data structures and their real-life applications.

This project can be further extended by adding advanced features such as shuffle mode, repeat functionality, favorite songs, and recently played tracks using stacks, making it more powerful and closer to real-world music systems.

Overall, this project highlights how core data structures can be effectively utilized to design efficient and user-oriented software systems.
